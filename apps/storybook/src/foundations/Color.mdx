# Color

UI Organized colour has two layers: a large **core palette** of ready-made OKLCH
ramps, and the **semantic tokens** that map a small set of roles onto that palette
for each theme. Components only ever speak in semantic roles; the core palette is
what those roles — and your custom themes — are built from.
